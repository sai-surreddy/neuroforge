# NeuroForge - AI Software Engineering Agent

Built for AMD Developer Hackathon 2026

## Setup

1. Start vLLM with Qwen model
2. Install: `pip install -r requirements.txt`
3. Run: `python app.py`
4. Open: `http://localhost:7860`

## Tech Stack
- AMD MI300X GPU
- Qwen2.5-Coder-14B-Instruct
- vLLM with ROCm
- Gradio UI
